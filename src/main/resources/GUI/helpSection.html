Welcome to the goBike application, your go to application for when you want to cycle around the city.

The application has many features, but to get started you should add some data.

----------------- 1.0 THE DATA PANEL -----------------

1.1 IMPORTING & EXPORTING DATA
++++++++++++++++++++++++++

From the FILE menu in the top left of the window you can IMPORT or EXPORT .csv files and view them in the raw data viewer. Make sure the files have the right headers for the object type they represent!
Once the data has been imported and added to a list it can be viewed in the DATA tab.
In the DATA tab you can select the type of data you wish to view (retailers/routes/wifi) and then select a stored list of those objects.


1.2 FAVOURITING
++++++++++++

Select any record and click the 'add to favourites' button to add it to your custom favourites list. You can view these favourites with the 'Show Favourite only', where you can remove them if you wish.
To return to the standard list click the 'Change to Default' button. Favourites are stored persistently between sessions just like the standard lists.


1.3 FILTERING AND SEARCHING
+++++++++++++++++++++++

The two boxes next to the FILTER button are used to filter data. The first identifies a header (Street / ZIP / Type etc) and the second displays all avaliable entries in the current list for that header.
Once both boxes have been selected, the FILTER button then displays all objects which match the criteria.

To search, select a header from the first box next to SEARCH, then enter a custom term in the empty box next to it and click SEARCH, and any objects which contain that term will be displayed.
To reset to the stand view click the 'Change to Default' button.


1.4 ALTERING DATA & ADDING AND DELETING ROUTES
++++++++++++++++++++++++++++++++++++++++++

To alter any parts of a certain record, click the SHOW DETAILS button and all infomation for a highlighted entry will be displayed in a pop up window. This is all fully editable and can be saved with the APPLY button.
If Routes are being viewed, in the bottom right they can be directly added and removed from a given list using the ADD ROUTE/DELETE ROUTE buttons.
Remember when adding a route to include the right kind of information for each entry!

----------------- 2.0 THE MAP PANEL -----------------

2.1 THE MAP VIEW
+++++++++++++

When the MAP panel is clicked, occasionally the map will not load. This is a known bug and can be fixed by right clicking and selecting 'Reload page'.
In the map view all Retailers and Wifi locations can be toggled for viewing, and highlighted routes and locations can have the closest wifi location/s shown.


2.2 FINDING ROUTES
++++++++++++++

Using the two search boxes in the top left, an address or set of co-ordinates can be entered in and Google's mapping algorithms will find a cycle route between the two after a brief delay.
If nothing changes then no route can be found, so ensure that the locations are spelt correctly.

----------------- 3.0 THE STAT PANEL -----------------

The statistics panel shows further details about a route. Select a highlighted route and click the "Show Stats" button to be redirected to the statistics panel.
The shown infomation will be about the highlighted route.
